#Data-Scientist Introduction#

##一、关于我们## ###1.1、Data-Scientist介绍 Data-Scientist是Lords组织在GitHub上的知识管理中心，也是传播MOOC

MOSS(大规模在线协同学习)，通过集体的智慧来加速学习进程，积累知识质量， 以及应用转化速度。这是对众筹概念的进一步发展，相信未来会越来越流行。 从Open API, Open Data, UGC, 到各种开源项目，集体写作，知识的呼 (开源)与吸(集体学习)将会呈现越来越明显的分布式协同性的特征， 这是我们所愿意看到的未来，也是我们所愿意贡献点滴的现在。

##二、项目环境配置##

###2.1 系统 推荐使用Linux 或者 Mac OS X，如果是虚拟机环境，建议使用CentOS6.4，方便与以后的大数据环境对接。

###2.2 开发工具###

  • R
  • Ruby 推荐使用rbenv管理Ruby版本
  • Rstudio
  • Git
  • Latex
  • Rubygems
  • Bundler
  • Jekyll 安装指南

###2.3 安装 Knitr 1. 打开Rstudio 2. 在console中输入install.packages(“knitr”); 3. 运行KnitPost.R脚本 4. KnitPost(“_drafts/heidsoft_testRmd.Rmd”) 执行后在项目的根目录下生成heidsoft_testRmd.md文件 5. rake post title="heidsoft_testRmd" 6. cp heidsoft_testRmd.md ./_posts/xxxxxx_heidsoft_testRmd.md

###2.4 Git常用方法 参见Git简明指南或者Github Help
